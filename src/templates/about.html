Project Background

Inspired by the paintings of Marko Rothko and Morris Louis, the goal of this application is to identify the time period of each artist's paintings using machine learning techniques. Initially, the goal was to use a neural network to predict the year painted for each painter's works, but this did was not the best model due to a need for a larger dataset.

How Does the Application Work?

The application is powered by a Python Flask application. Machine learning models were explored using Python's skikit-learn library to extract specific metrics from each artist's paintings and then test the relationships between these metrics.
